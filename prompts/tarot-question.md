# 타로 질문 입력 프롬프트

## tarot-question

### 본문 3줄 미리보기

```text
타로 사이트에 질문 입력 화면을 만들어 주세요.
사용자가 고른 주제에 맞춰 궁금한 내용을 한 문장으로 적을 수 있게 해 주세요.
입력칸과 다음 버튼은 휴대전화에서도 크게 보이게 해 주세요.
```

### 복사용 전문

```text
타로 사이트에 질문 입력 화면을 만들어 주세요.
사용자가 고른 주제에 맞춰 궁금한 내용을 한 문장으로 적을 수 있게 해 주세요.
입력칸과 다음 버튼은 휴대전화에서도 크게 보이게 해 주세요.

질문 입력 화면에는 선택한 주제가 위쪽에 보이면 좋겠습니다.
입력칸에는 예시 문장을 넣어 주세요.
예시는 `오늘 제가 가장 신경 써야 할 것은 무엇인가요?`처럼 초보자가 따라 쓰기 쉬운 문장으로 해 주세요.
질문을 비워 둔 상태에서 다음으로 넘어가려 하면 부드럽게 안내해 주세요.
문구는 겁주지 말고 친절하게 작성해 주세요.
완성 후에는 입력칸, 예시 문장, 다음 버튼을 확인할 기준을 알려 주세요.
```

### 사용 위치

- 원고 파일: `03_chapter3_current.md`
- 장: Chapter 3
- 위치: 질문 입력 화면 만들기
