# 타로(tarot) 사이트 기본 화면 프롬프트(prompt)

## tarot-card-sample-1

### 본문 3줄 미리보기

```text
타로(tarot) 사이트에 사용할 샘플 카드 이미지를 1장 만들어 주세요.
신비롭지만 무섭지 않고, 초보자도 편하게 볼 수 있는 분위기로 해 주세요.
나중에 같은 스타일의 카드 세트로 확장할 수 있게 기준을 함께 정리해 주세요.
```

### 복사용 전문

```text
타로(tarot) 사이트에 사용할 샘플 카드 이미지를 1장 만들어 주세요.

카드 주제는 `The Fool`로 해 주세요.
세로형 카드 비율로 만들어 주세요.

너무 어둡거나 공포스럽지 않게 해 주세요.
완성 후에는 이 카드의 색감, 테두리, 일러스트 스타일을 짧게 설명해 주세요.
이미지 생성 한도가 있을 수 있으니, 한 번에 너무 많은 변형을 요청하지 말아 주세요.
```

## tarot-design

### 본문 3줄 미리보기

```text
타로(tarot)카드 리딩 웹사이트의 전체 디자인 방향을 먼저 정해 주세요.
초보자도 편하게 볼 수 있게 신비롭지만 무섭지 않은 분위기로 해 주세요.
휴대전화에서도 읽기 좋은 화면을 기준으로 제안해 주세요.
```

### 복사용 전문

```text
타로(tarot)카드 리딩 웹사이트의 전체 디자인 방향을 먼저 정해 주세요.
초보자도 편하게 볼 수 있게 신비롭지만 무섭지 않은 분위기로 해 주세요.
휴대전화에서도 읽기 좋은 화면을 기준으로 제안해 주세요.

사이트 이름은 `오늘의 카드 리딩`으로 하겠습니다.
사용자는 주제를 고르고, 질문을 입력하고, 카드 3장을 뽑고, 결과 해석을 봅니다.
전체 분위기는 차분하고 따뜻해야 합니다.
글자는 읽기 쉬워야 하고, 버튼은 누르기 쉬워야 합니다.
카드 이미지는 제공 자료를 사용할 예정이므로 이미지가 깨지지 않게 다루는 기준도 포함해 주세요.
먼저 색상, 글자 느낌, 버튼 스타일, 화면 흐름을 짧게 정리해 주세요.
그 다음 실제 제작에 들어갈 준비가 되었는지 체크리스트를 알려 주세요.
```

### 사용 위치

- 원고 파일: `07_tarot_build.md`

## tarot-main

### 본문 3줄 미리보기

```text
타로(tarot) 사이트의 메인 화면을 만들어 주세요.
처음 들어온 사람이 어떤 사이트인지 바로 알 수 있게 해 주세요.
시작 버튼을 누르면 주제 선택으로 이어지는 구조로 만들어 주세요.
```

### 복사용 전문

```text
타로(tarot) 사이트의 메인 화면을 만들어 주세요.
처음 들어온 사람이 어떤 사이트인지 바로 알 수 있게 해 주세요.

메인 화면에는 큰 제목, 짧은 소개 문장, 시작 버튼이 필요합니다.
분위기는 신비롭지만 부담스럽지 않게 해 주세요.

사용자가 무엇을 누르면 되는지 바로 알 수 있어야 합니다.
완성 후에는 제가 확인할 항목을 3가지로 정리해 주세요.
```

## tarot-card-detail

### 본문 3줄 미리보기

```text
타로(tarot) 결과 화면에서 카드를 누르면 자세한 설명이 보이게 해 주세요.
초보자도 이해할 수 있게 카드 의미를 짧고 부드럽게 보여 주세요.
팝업은 휴대전화에서도 닫기 버튼이 잘 보이게 해 주세요.
```

### 복사용 전문

```text
타로(tarot) 결과 화면에서 카드를 누르면 자세한 설명이 보이게 해 주세요.
초보자도 이해할 수 있게 카드 의미를 짧고 부드럽게 보여 주세요.
팝업은 휴대전화에서도 닫기 버튼이 잘 보이게 해 주세요.

카드 상세 보기에는 카드 이름, 핵심 의미, 오늘의 조언 1문장이 있으면 좋겠습니다.
문장은 너무 단정적이거나 불안하게 쓰지 말고, 선택을 도와주는 부드러운 조언처럼 써 주세요.
사용자가 팝업을 닫고 다시 결과 화면으로 돌아올 수 있어야 합니다.
카드 이미지가 작게 깨져 보이지 않게 해 주세요.
완성 후에는 카드 상세 보기를 확인하는 방법을 알려 주세요.
```

### 사용 위치

- 원고 파일: `07_tarot_build.md`

## tarot-history

### 본문 3줄 미리보기

```text
타로(tarot) 사이트에 최근 리딩 이력을 간단히 보여 주는 영역을 추가해 주세요.
너무 복잡한 저장 기능이 아니라, 사용자가 방금 본 결과를 다시 확인하는 정도면 됩니다.
비어 있을 때도 자연스러운 안내 문장이 보이게 해 주세요.
```

### 복사용 전문

```text
타로(tarot) 사이트에 최근 리딩 이력을 간단히 보여 주는 영역을 추가해 주세요.
너무 복잡한 저장 기능이 아니라, 사용자가 방금 본 결과를 다시 확인하는 정도면 됩니다.
비어 있을 때도 자연스러운 안내 문장이 보이게 해 주세요.

최근 이력에는 주제, 질문 일부, 선택한 카드 이름, 짧은 결과 요약이 보이면 좋겠습니다.
휴대전화에서도 카드처럼 보기 좋게 정리해 주세요.
이력이 없을 때는 `아직 저장된 리딩이 없습니다`처럼 부드럽게 안내해 주세요.
사용자가 다시 리딩하기 버튼을 쉽게 찾을 수 있게 해 주세요.
완성 후에는 최근 이력이 잘 보이는지 확인할 기준을 알려 주세요.
```

### 사용 위치

- 원고 파일: `07_tarot_build.md`
- 장: Chapter 6
- 페이지: 17페이지
