```text

이 책에 나오는 긴 프롬프트(prompt)는 직접 외우거나 다시 입력하지 않으셔도 됩니다.
```

## prompt-basic-structure

### 본문 3줄 미리보기

```text
앞으로 제가 요청할 때는 목적, 참고할 재료, 원하는 결과를 함께 말하겠습니다.
제가 초보자라는 점을 기준으로 답해 주세요.
결과가 나오면 다음에 확인할 것도 짧게 알려 주세요.
```

### 복사용 전문

```text
앞으로 제가 요청할 때는 목적, 참고할 재료, 원하는 결과를 함께 말하겠습니다.
제가 초보자라는 점을 기준으로 답해 주세요.
결과가 나오면 다음에 확인할 것도 짧게 알려 주세요.

저는 웹사이트 제작을 처음 해 봅니다.
제가 원하는 결과를 말하면, ChatGPT 대화창 안에서 따라 할 수 있는 방식으로 안내해 주세요.
설명은 한 번에 너무 길게 하지 말고, 제가 바로 확인할 수 있는 다음 행동을 함께 알려 주세요.
결과물이 마음에 들지 않을 때 다시 고칠 수 있는 문장도 같이 제안해 주세요.
```

## first-site

### 본문 3줄 미리보기

```text
첨부한 이미지와 어울리는 반응형 웹사이트를 만들어 주세요.
휴대전화와 컴퓨터에서 모두 보기 좋게 구성해 주세요.
처음 방문한 사람이 무엇을 보면 되는지 바로 알 수 있게 해 주세요.
```

### 복사용 전문

```text
첨부한 버블 공연 사진과 어울리는 반응형 웹사이트를 만들어 주세요.
휴대전화와 컴퓨터에서 모두 보기 좋게 구성해 주세요.
처음 방문한 사람이 무엇을 보면 되는지 바로 알 수 있게 해 주세요.

저는 코딩을 직접 수정할 줄 모르는 초보자입니다.
첨부한 이미지는 야외에서 비눗방울 공연을 보는 장면입니다.
이 사진을 참고해서 한 페이지짜리 소개 웹사이트를 만들어 주세요.
사이트에는 큰 제목, 짧은 소개 문장, 이미지가 돋보이는 영역, 행동을 유도하는 버튼 1개가 있으면 좋겠습니다.
분위기는 밝고 깔끔하게 잡아 주세요.
너무 복잡한 기능은 넣지 말고, 처음 보는 사람도 편하게 읽을 수 있게 해 주세요.
결과는 나중에 GitHub Pages로 올릴 수 있는 웹사이트 형태로 정리해 주세요.
완성 후에는 제가 확인해야 할 항목을 짧은 체크리스트로 알려 주세요.
```

### 사용 위치

- 원고 파일: `03_first_site.md`
- 장: Chapter 2
- 페이지: 4페이지
