# 타로 결과 화면 프롬프트

## tarot-result

### 본문 3줄 미리보기

```text
타로 사이트에 결과 화면을 만들어 주세요.
선택한 카드 3장과 각 카드의 의미, 전체 조언을 보기 좋게 보여 주세요.
문장은 무섭거나 단정적이지 않고 부드러운 조언처럼 작성해 주세요.
```

### 복사용 전문

```text
타로 사이트에 결과 화면을 만들어 주세요.
선택한 카드 3장과 각 카드의 의미, 전체 조언을 보기 좋게 보여 주세요.
문장은 무섭거나 단정적이지 않고 부드러운 조언처럼 작성해 주세요.

결과 화면에는 사용자가 고른 주제와 질문이 다시 보이면 좋겠습니다.
카드 3장은 각각 카드 이름, 짧은 의미, 오늘의 조언으로 정리해 주세요.
마지막에는 전체 흐름을 요약하는 문장을 넣어 주세요.
다시 뽑기 버튼과 처음으로 돌아가기 버튼도 넣어 주세요.
휴대전화에서 카드와 해석 문장이 너무 좁게 보이지 않게 해 주세요.
완성 후에는 결과 화면에서 확인해야 할 항목을 체크리스트로 알려 주세요.
```

### 사용 위치

- 원고 파일: `07_tarot_build.md`
- 장: Chapter 6
- 페이지: 13페이지

## tarot-mobile-fix

### 본문 3줄 미리보기

```text
타로 사이트를 휴대전화에서 보기 좋게 다듬어 주세요.
글자, 버튼, 카드 간격이 작거나 겹치지 않게 조정해 주세요.
가로로 밀어야 보이는 화면이 생기지 않게 해 주세요.
```

### 복사용 전문

```text
타로 사이트를 휴대전화에서 보기 좋게 다듬어 주세요.
글자, 버튼, 카드 간격이 작거나 겹치지 않게 조정해 주세요.
가로로 밀어야 보이는 화면이 생기지 않게 해 주세요.

저는 휴대전화에서 이 사이트를 확인하고 있습니다.
메인 화면, 주제 선택, 질문 입력, 카드 뽑기, 결과 화면을 모두 모바일 기준으로 점검해 주세요.
버튼은 손가락으로 누르기 쉬워야 합니다.
카드 이미지는 너무 작아지지 않아야 합니다.
결과 문장은 줄 간격이 답답하지 않아야 합니다.
수정 후에는 모바일에서 다시 확인할 항목을 5개 이내로 알려 주세요.
```

### 사용 위치

- 원고 파일: `07_tarot_build.md`
- 장: Chapter 6
- 페이지: 19페이지
