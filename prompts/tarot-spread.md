# 타로(tarot) 카드 뽑기 프롬프트(prompt)

## tarot-spread

### 본문 3줄 미리보기

```text
타로(tarot) 사이트에 카드 뽑기 화면을 만들어 주세요.
사용자가 여러 카드 중 3장을 선택하면 선택된 상태가 분명히 보이게 해 주세요.
카드는 휴대전화에서도 누르기 쉬운 크기와 간격으로 배치해 주세요.
```

### 복사용 전문

```text
타로(tarot) 사이트에 카드 뽑기 화면을 만들어 주세요.
사용자가 여러 카드 중 3장을 선택하면 선택된 상태가 분명히 보이게 해 주세요.
카드는 휴대전화에서도 누르기 쉬운 크기와 간격으로 배치해 주세요.

카드 뽑기 화면은 질문 입력 다음에 나옵니다.
처음에는 카드 뒷면이 보이고, 사용자가 누르면 선택된 카드처럼 표시되면 좋겠습니다.
3장을 선택하기 전에는 결과 보기 버튼이 너무 먼저 강조되지 않게 해 주세요.
3장을 모두 선택하면 결과 보기 버튼이 자연스럽게 눈에 들어오게 해 주세요.
카드가 너무 작거나 서로 붙어 있지 않게 해 주세요.
완성 후에는 카드 선택 전, 1장 선택 후, 3장 선택 후 상태를 확인하는 기준을 알려 주세요.
```

- 장: Chapter 3
