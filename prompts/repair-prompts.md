## repair-pages-url

### 본문 3줄 미리보기

```text
GitHub Pages 주소가 바로 열리지 않습니다.
기다리면 되는 상황인지, 설정을 다시 확인해야 하는 상황인지 구분해 주세요.
ChatGPT와 GitHub 연동으로 먼저 해결할 수 있는 방법을 알려 주세요.
```

### 복사용 전문

```text
GitHub Pages 주소가 바로 열리지 않습니다.
기다리면 되는 상황인지, 설정을 다시 확인해야 하는 상황인지 구분해 주세요.
ChatGPT와 GitHub 연동으로 먼저 해결할 수 있는 방법을 알려 주세요.

현재 상황:
- 받은 주소: [여기에 주소를 붙여 넣어 주세요.]
- 보이는 화면: [예: 404 화면, 빈 화면, 예전 화면, 로딩 중]

저는 GitHub를 잘 모르는 초보자입니다.
먼저 몇 분 기다리면 되는 상황인지 알려 주세요.
그 다음 확인할 항목을 쉬운 순서로 알려 주세요.
제가 GitHub 화면을 직접 봐야 한다면, 눌러야 할 위치를 짧게 안내해 주세요.
최종 목표는 휴대전화와 컴퓨터에서 같은 최신 사이트가 열리는 것입니다.
```

### 사용 위치

- 원고 파일: `04_chapter4_current.md`
- 장: 부록
- 위치: GitHub Pages 주소가 안 뜰 때
