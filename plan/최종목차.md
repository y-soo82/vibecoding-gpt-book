# 바이브코딩 1차 GPT 입문서 최종 목차

## 목차 설계 원칙

- 독자가 첫 번째 성공을 빨리 경험하도록 구성합니다.
- ChatGPT 메뉴 설명은 실습에 필요한 만큼만 먼저 다룹니다.
- GitHub 배포는 GitHub 화면 조작 중심이 아니라 ChatGPT GitHub 연동 대화 중심으로 설명합니다.
- 타로 사이트는 완성 화면을 먼저 보여 준 뒤 따라 만들게 합니다.
- 긴 프롬프트는 읽는 글이 아니라 복사해서 실행하는 도구로 취급합니다.
- 각 장은 이미지가 먼저 나오고, 글은 이미지를 따라가는 짧은 안내가 되게 합니다.
- 전문가용 도구 설명보다 ChatGPT 대화창에서 따라 하는 흐름을 우선합니다.
