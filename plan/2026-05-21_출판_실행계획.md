# 2026-05-21 출판 실행 계획

## 목표

바이브코딩 1차 GPT 입문서를 GitHub Pages에서 바로 확인할 수 있는 반응형 웹 전자책으로 정리하고, Sigil EPUB 전환을 위한 초안 패키지를 생성합니다.

## 기준

- 책은 4챕터 구조로 정리합니다.
- Chapter 2의 첫 사이트 입력 이미지는 `projects/e-book/docs/첫사이트만들기.jpeg`를 기준으로 합니다.
- 출판용 웹 자산은 EXIF와 GPS 정보를 제거한 복사본만 사용합니다.
- Chapter 3의 타로앱은 현재 org 기준 HTML 결과물을 기준 결과물로 사용합니다.
- 타로앱은 메이저 22장 선택 구조, 정방향과 역방향, 5영역 리딩, 최근 이력 5건 저장 기준을 반영합니다.
- ChatGPT와 GitHub 로그인 화면은 개인정보와 권한 화면 문제 때문에 최종 출판 전 실제 캡처 또는 명시적 예시 화면 판정이 필요합니다.

## 이번 실행 범위

1. 첫 사이트 예제를 새 기준 이미지에 맞게 갱신합니다.
2. 웹북을 Chapter 1/2 확인본에서 4챕터 확인본으로 확장합니다.
3. Chapter 3과 Chapter 4 원고를 웹북 로더에 연결합니다.
4. 대표 캡처를 다시 생성합니다.
5. EPUB 초안 패키지를 생성합니다.
6. 로컬 브라우저 기준으로 웹북, 첫 사이트, 타로 사이트가 열리는지 확인합니다.
