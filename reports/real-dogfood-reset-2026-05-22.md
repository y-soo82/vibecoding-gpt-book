# 실제 독자 따라하기 재검수 리셋

작성일: 2026-05-22

## 정정

기존 `reports/reader-dogfood-gpt-run-2026-05-22.md`와 `reports/superbrain-reader-dogfood-analysis-2026-05-22.md`는 실제 ChatGPT에 책의 프롬프트를 입력하고, 실제 GitHub 화면을 따라가며, 실제 배포 URL까지 확인한 기록이 아닙니다.

해당 문서는 로컬 HTML, 예제 사이트, 프롬프트 구조를 점검한 제한적 검수 기록입니다. 따라서 “독자가 실제로 따라 해서 성공했다”는 근거로 사용할 수 없습니다.

## 리셋 사유

- Git/GitHub 화면 중 다수가 실제 웹 화면 캡처가 아니라 시뮬레이션 또는 대체 이미지입니다.
- ChatGPT 입력, 이미지 첨부, 답변 생성, 결과 확인 흐름을 실제 로그인 세션에서 끝까지 실행하지 않았습니다.
- 타로(tarot) 화면은 실제 GPT 결과물이 아니라 로컬 예제 사이트 또는 출판용 화면입니다.
- 기존 dogfood 문서는 실제 독자의 진행 기록이 아니라 내부 구조 점검 문서에 가깝습니다.

## 새 재작업 기준

1. 실제로 실행하지 않은 항목을 실행했다고 쓰지 않습니다.
2. 실제 웹 화면이 아닌 이미지를 실제 캡처라고 부르지 않습니다.
3. 현재 웹북 이미지는 출처별로 분류하고, 교체 필요 이미지는 별도 목록으로 관리합니다.
4. ChatGPT 실제 실행은 로그인된 Chrome 또는 별도 Playwright 로그인 프로필에서 진행합니다.
5. 로그인이나 권한 때문에 막히면 그 지점을 그대로 진행 기록에 남깁니다.
6. 실제 실행 기록을 바탕으로만 슈퍼브레인 분석과 전자책 수정을 진행합니다.

## 현재 실제 실행 상태

별도 Playwright/Chrome 프로필로 `https://chatgpt.com/` 접속을 확인했습니다. 화면은 로그인 필요 상태였습니다. 따라서 이번 단계에서는 실제 프롬프트 입력, 이미지 첨부, ChatGPT 결과 생성, GitHub 연결 배포를 완료하지 못했습니다.
