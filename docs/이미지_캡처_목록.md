## 전체 기준

- 현재 출판 확인본 기준 수량은 50장입니다.
- 중복 제거를 우선하므로, 이미지를 60~70장으로 억지로 채우지 않습니다.
- Chapter 1에는 결과물 이미지를 넣지 않습니다.
- 한 번 나온 이미지 파일은 다른 장면에서 다시 사용하지 않습니다.
- 프롬프트(prompt) 박스는 HTML(HyperText Markup Language) 컴포넌트로 보여 주며, 같은 프롬프트(prompt)를 이미지로 다시 보여 주지 않습니다.
- 결과물은 ChatGPT 대화창 안 미리보기가 아니라 실제 브라우저(browser) 또는 배포 URL(Uniform Resource Locator)에서 확인한 화면으로 캡처(capture)합니다.
- 개인정보가 보이는 화면은 출판용 편집본에서 제거합니다.

## 품질 기준

- 이미지 파일 경로는 책 전체에서 한 번만 사용합니다.
- 프롬프트(prompt) 박스는 본문 HTML(HyperText Markup Language) 컴포넌트로 처리하고, 같은 내용을 이미지로 중복하지 않습니다.
- HTML(HyperText Markup Language) 결과 이미지는 실제 브라우저(browser) 또는 배포 URL(Uniform Resource Locator)에서 확인한 화면만 사용합니다.
- Chapter 2와 Chapter 3의 결과 이미지는 책의 기능, 화면 흐름, UI(User Interface), UX(User Experience) 검수 기준이 됩니다.
- 데스크톱(desktop) 컷과 모바일(mobile) 컷은 서로 다른 목적을 가져야 합니다.
- 캡처(capture) 안 글자가 모바일(mobile)에서 읽히지 않으면 컷을 분할합니다.
- 개인정보가 보이는 화면은 출판용 편집본에서 제거합니다.
